# プレゼント交換アプリ

## 全体概要
複数人でのプレゼント交換の際に、交換相手をランダムに決定するためのWebアプリです。

## Versions
- v1.0-frontend : フロント技術のみで作成した初期版（Qiita記事）

## v1.0-frontend について

### URL
https://desuharudesu.github.io/presentExchangeApp/index.html

### 概要
最低限の機能を実装したバージョンです。  
幹事が参加者の名前を入力すると、参加人数分のURLが発行されます。

- 幹事が参加者の名前を入力
- シャッフル後、人数分のURLを発行
- 幹事が各参加者に該当URLを送付
- URLを開くと、誰にプレゼントを渡すかが表示されます

### 使用技術
#### フロントエンド
- HTML5  
- CSS3  
- JavaScript（Vanilla JS）
