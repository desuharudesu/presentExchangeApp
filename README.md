# プレゼント交換アプリ
## 全体概要
複数人でのプレゼント交換の際、交換相手をシャッフルするウェブアプリです。
### Ver1について
#### url
https://desuharudesu.github.io/presentExchangeApp/index.html
#### 概要
最低限の機能を実装。
幹事が参加する人の名前を入力すると、人数分のurlが発行されます。
urlは幹事が手動で該当参加者に送付。
urlを開くと誰にプレゼントを渡すのか記載されています。
#### 使用技術一覧
フロント：
-HTML5
-CSS3
-Javascript
